# Flutter Builder Plugin for Acode

A simple plugin for Acode editor to build and run Flutter projects directly from mobile.

## Features
- Run Flutter project
- Build APK
- Run Flutter commands from Acode

## Installation

1. Open Acode
2. Go to Plugins
3. Click Install from source
4. Paste this URL:

https://raw.githubusercontent.com/YaminDeveloper/flutter-builder-plugin/main/manifest.json

## Files

- manifest.json
- main.js
- icon.svg
